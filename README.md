# Canada's Labour Market Under Pressure — Employment Analysis (2019–2026)

## Project Overview
This project analyzes Canada's labour market trends from January 2019 to April 2026 using official Statistics Canada data. It covers employment patterns across 10 provinces before, during, and after the COVID-19 pandemic, including the impact of the 2025 trade shock.

## Tools Used
- Python (Pandas) — data cleaning
- MySQL — data storage and SQL analysis
- Power BI — interactive dashboard (coming soon)
- Git/GitHub — version control

## Data Source
Statistics Canada — Table 14-10-0287-03
[Labour force characteristics by province, monthly, seasonally adjusted](https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1410028703)

## Dataset
- 5,280 rows
- 10 provinces
- 6 metrics: Employment, Full-time employment, Part-time employment, Unemployment rate, Participation rate, Employment rate
- January 2019 to April 2026
